# Project Overview
This project analyzes Walmart sales data using Microsoft Excel to derive meaningful business insights.
The objective of this analysis is to clean, transform, and visualize raw sales data to support data-driven decision-making.

## The project demonstrates strong skills in:
Data Cleaning
Data Analysis
Pivot Tables
Dashboard Creation
Business Insight Generation

 ## Tools Used
Microsoft Excel
Pivot Tables
Power Query 
Charts & Dashboard
Excel Formulas (VLOOKUP/XLOOKUP, IF, SUMIFS, COUNTIFS, etc.)

## Data Cleaning Process

Removed duplicate records
Handled missing/null values
Standardized date formats
Corrected inconsistent text values
Created calculated columns
Verified data types

## Key Analysis Performed

 Total Sales Analysis

 Profit Analysis

 Sales by Region

 Sales by Category
 
 Top Performing Products

 Profit vs Sales relation

## Dashboard Features

The Excel dashboard includes:

KPI Cards (Total Sales, Total Profit, Total Orders , total profit margin , total quantity)

Region-wise Sales Chart

Category-wise Performance

Monthly Trend Chart

Interactive Filters / Slicers

 ## Key Insights

Identified top-performing regions contributing maximum revenue

Found categories generating high sales but low profit

Observed monthly sales trends and seasonal patterns

Highlighted underperforming areas for improvement

## Business Impact

This analysis helps:

Improve regional sales strategy

Optimize product category focus

Increase profitability

Support data-driven business decisions
